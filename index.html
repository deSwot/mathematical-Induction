Mathematical Induction Proof
Mohammad Nasir

Question\[\sum_{j=1}^n j = \frac{n(n+1)}{2} \]Base case n = 1\[\sum_{j=1}^1 1 = \frac{1(1+1)}{2} \]\[1=\frac{1(1+1)}{2}\]
Assume for some c = n\[\sum_{j=1}^c j = \frac{c(c+1)}{2} \]Show true for c + 1\[\sum_{j=1}^{c + 1} j = \frac{(c + 1)((c + 1)+1)}{2} \]
